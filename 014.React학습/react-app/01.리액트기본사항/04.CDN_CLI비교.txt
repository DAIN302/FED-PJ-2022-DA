[ 리액트 CDN 방식 / CLI 환경방식 차이 ]
1. import JS사용시 CDN은 확장자 생략불가
2. 다른 외부 JS import시 CDN은 본 html에도 표기
3. render 메서드 사용 시 CDN은 ReactDOM.render()를 바로 사용했으나 
   CLI방식은 createRoot() 를 사용하여 2단계로 렌더링 코드 작성해야함 
   예) const root = ReactDOM.createRoot(document.getElementById('root'));
        root.render(<App/>); 
4. <></> 빈 프레그먼트 요소는 CLI 에서만 지원
5. CDN은 앞에 모듈을 반드시 써야함, CLI에서는 쓰지않아도 됨 (import해서)   
   예) CDN -> <React.Fragment>, React.useState
       CLI -> <Fragment>, useState
