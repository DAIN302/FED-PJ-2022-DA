1. 리액트 앱 설치
- 설치위치 : 04.DC_PJ\02.구현소스 
- 설치이름 : dc-app
- 스크립트 : npx create-react-app dc-app 
npx는 npm 과 달리 버전 선택하지 않아도 
최신버전의 리액트 모듈을 설치해 준다!

2. 패키지 모듈 설치
- 설치위치 : 04.DC_PJ\02.구현소스\dc-app
- 스크립트 :
  (1) npm i react-router-dom
  (2) npm i jquery

3. 설치확인 
- 파일명 : package.json
- 항목 : "dependencies"

4. 개발서버구동
- 서버구동위치 : 04.DC_PJ\02.구현소스\dc-app
- 스크립트 :
  (1) 시작 : npm start
  (2) 멈춤 : ctrl + c 후 질문에 답 : y
