[ 모듈별 정의 : 구조정의/객체화/기능정의 ]

[1] 배너모듈

[2] 메뉴버튼모듈
    - 모듈명 : MenuBtn(JS/CSS/data)
   (1) 구조정의
        Root > section >img Box + title Box + button Box
   (2) 객체화
    {
        "isrc" : "이미지경로",
        "tit" : "작은타이틀^큰타이틀",
        "btn" : "버튼문구"
        "link" : "이동카테고리"
    }
    -> "tit" 속성은 ^로 데이터 구분
    -> 컴포넌트별 객체화 데이터 반영은 기본

   (3) 기능정의
       1) 버튼 오버 시 하이라이트
          (mouseover/mouseout) + css "on"정의
       2) 버튼 클릭 시 페이지이동 
          (click) + location(라우트 이동) 

[3] 비디오 소개 모듈
    - 모듈명 : VidIntro(JS/CSS/data)
   (1) 구조정의 :
       Root > video box(>video+img) + title box(>heading + p tag) 
   (2) 객체화 :
       {
            "vsrc" : "유튜브동영상경로",
            "stit" : "작은타이틀",
            "btit" : "큰타이틀",
            "sum" : "요약소개",
            "desc" : "설명",
            "link" : "관련링크",
       } 
   (3) 기능정의
       1) 동영상 영역 클릭 시 동영상 재생/멈춤
       2) 관련링크 새창열기

[4] 비디오 스와이프 모듈
    - 모듈명 : VidSwipe(JS/CSS/data)
   (1) 구조정의 :
       스와이퍼 모듈 + 비디오재생모듈
   (2) 객체화 :
       {
            isrc : "이미지경로",
            vsrc : "유튜브동영상경로",
            cat : "카테고리",
            tit : "타이틀",
       } 
   (3) 기능정의 :
       1) 비디오 메뉴 스와이프 기능(스와이퍼)
       2) 비디오 메뉴 클릭 시 별창 동영상 재생기능(닫기버튼)

[5] 캐릭터 모듈
    - 모듈명 : Cat(JS/CSS/data)
   (1) 구조정의 :
       제목 + 스와이퍼 모듈
       -> 하위 스와이퍼 모듈(이미지+이름)
   (2) 객체화 :
       {
            idx : "인덱스번호(유일키)",
            tmsrc : "썸네일이미지",
            cname : "캐릭터이름",
            cdesc : "캐릭터설명",
            facts : "캐릭터상세설명",
            alignment : "캐릭터종류(히어로/빌런/복잡)",
            type : "캐릭터타입(개인/단체)"
       } 
   (3) 기능정의 :
       1) 캐릭터 메뉴 스와이프 기능(스와이퍼)
       2) 캐릭터 메뉴 클릭 시 캐릭터 상세페이지 이동(라우터 사용)

[6] 상세페이지 모듈 : 페이지구성 모듈
    - 모듈명 : Detail(JS/CSS/data)
   (1) 구조정의 :
       배너모듈 + 캐릭터명 + 캐릭터소개 + 캐릭터상세설명 + 관련캐릭터
       
   (2) 객체화 : 
       - 캐릭터모듈 데이터 사용
       - 배너모듈 데이터 사용
        
   (3) 기능정의 :
       1) 캐릭터 메뉴에서 넘어온 캐릭터 키를 가지고 데이터값 세팅

[7] 검색 모듈 : 페이지구성 모듈
    - 모듈명 : Search(JS/CSS/data)
   (1) 구조정의 :
       검색옵션 모음 + 캐릭터 리스트 모듈
       
   (2) 객체화 : 
       - 캐릭터모듈 데이터 사용
        
   (3) 기능정의 :
       1) 검색 전 전체 캐릭터 리스트 보여주기
       2) 검색 후 조건에 맞는 캐릭터 리스트 보여주기
       3) 검색 조건 : cat_data의 구성 항목
          (1) 캐릭터 이름 : cname
          (2) 캐릭터 분류 : alignment(히어로/빌런/복잡)
          (3) 캐릭터 타입 : type(개인/단체)

[8] 캐릭터 리스트 모듈 : 개별 모듈
    - 모듈명 : CatList(JS/CSS/data)
   (1) 구조정의 :
       캐릭터 이미지 + 캐릭터 이름
       
   (2) 객체화 : 
       - 캐릭터모듈 데이터 사용 (Search 모듈에서 전달 )
        
   (3) 기능정의 :
       - 캐릭터 데이터 배열값에 따른 리스트 구성 
       - 클릭 시 캐릭터 상세페이지로 이동(Detail.js)
